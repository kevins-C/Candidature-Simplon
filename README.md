# Candidature-Simplon
Projet pour la formation CDA
Le développement technologique est ma passion. Aujourd'hui tout se fait en ligne et devenir développeur a toujours été mon rêve. Je souhaite apprendre l'éco-conception pour créer un numérique plus responsable.
En rejoignant ce programme en partenariat avec BPCE SI, je veux apprendre à construire ce monde numérique de manière responsable. Mon objectif est de créer des applications performantes qui respectent l'environnement en étant optimisées et moins énergivores.
